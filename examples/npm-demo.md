# 45–60 Second npm Demo

```text
AI coding agents use terminals designed for humans.
AgentShell gives them compact, evidence-preserving observations.

npm install -g @yakshithk/agentshell
aish init
aish inspect
aish tree
aish search "login"
aish diff
aish test -- npm test

The installed rules teach Claude Code, Codex, Cursor, and OpenCode to prefer
these bounded commands over noisy recursive listings, file dumps, patches, and
test logs. Summary first; exact paths, lines, errors, commands, and exit codes
remain available when the agent needs detail.
```
